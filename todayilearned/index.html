#todayilearned feed


An automatic feed of #todayilearned reflections from the CodeBuddies independent code learners community. Share something you learned -- no matter how small -- by including the #todayilearned hashtag anywhere on our Slack.

Help improve this page by giving feedback in the #codebuddies-meta channel on Slack.

#todayilearned is a sister project of the #100daysofcode challenge, which challenges you to share one new thing you learn that's code-related -- no matter how small -- each day. It could be a code snippet. It could be a summary of a stackoverflow answer. It could be a sentence referencing an "aha!" moment. Let's celebrate these small wins.

What is CodeBuddies? We're a community of independent code learners who support each other on Slack , schedule hangouts to learn with each other, and share shout-outs to stuff we think is interesting or links to projects we're working on in a recurring anonymous community-wide newsletter. Learning with each other helps us learn faster. We've also teamed up with The Odin Project volunteers on a Facebook group. We strive to create a safe space for anyone motivated to improve their coding skills. The project is free and open-sourced on Github.

Is CodeBuddies an organization? No. It's a free-spirited community of enterprising learners and amazing volunteers who share knowledge with each other.
