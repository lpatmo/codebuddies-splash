#todayilearned feed


An automatic feed of #todayilearned reflections from the CodeBuddies independent code learners community. Share something you learned -- no matter how small -- by including the #todayilearned hashtag in any channel on our Slack.

Help improve this page by giving feedback in the #codebuddies-meta channel on Slack.

#todayilearned is a sister project of the #100daysofcode challenge, which challenges you to share one new thing you learn that's code-related -- no matter how small -- each day. It could be a code snippet. It could be a summary of a stackoverflow answer. It could be a sentence referencing an "aha!" moment.

What is CodeBuddies? We're a community of independent code learners who support each other on Slack , schedule hangouts to learn with each other, and contribute to an anonymous weekly newsletter. Learning with each other helps us learn faster. We've also teamed up with The Odin Project member volunteers on a Facebook group. We strive to create a safe space for anyone interested in improving their coding skills. The project is free and open-sourced on Github.

Is CodeBuddies an organization? No. It's a free-spirited community of enterprising learners and amazing volunteers who enjoy sharing knowledge with each other.
